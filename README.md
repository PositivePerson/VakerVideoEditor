## Deploy

[Deployment on Vercel](https://vercel.com/kfischer/frontend)

## How to use it?

Install dependencies:

```bash
    npm install
    # or
    yarn
```

Start Local Next-Server

```bash
    npm run dev
    # or
    yarn dev
```
